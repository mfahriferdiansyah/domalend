# Crediflex AVS Contracts

## Usage

### Build

```shell
$ forge build
```

### Deploy Eigen

```shell
$ forge script script/DeployEigenLayerCore.s.sol --rpc-url <RPC_URL> --broadcast
```

### Deploy Crediflex

```shell
$ forge script script/CrediflexDeployer.s.sol --rpc-url  <RPC_URL> --broadcast
```
